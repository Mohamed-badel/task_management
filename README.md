
# Task Management App

A full-stack task manager built with React and Supabase.

## Features
- Authentication (Login/Signup)
- Create, update, delete tasks
- Protected routes
- Supabase database

## Tech Stack
- React
- Supabase
- Tailwind CSS

## Installation

git clone https://github.com/Mohamed-badel/task_management.git
cd task_management
npm install
npm run dev

## Environment Variables

Create a .env file:

VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
